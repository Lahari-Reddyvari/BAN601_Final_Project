# Sales Performance Dashboard for Accessories

## Project Overview
The Sales Performance Dashboard for Accessories is a Streamlit application that lets users filter and visualize sales data for handbags and wallets for different years.
The dashboard helps analyze sales trends and performance through interactive charts and summary metrics.

## Features
- Monthly Sales Trend Chart
- Cumulative Sales Trend Chart
- Summary Statistics Table
- Total Sales Metric
- Sidebar Filters for:
  Year
  Product
- Color coded line plots for each product
- Distribution of Sales data visualization 

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## Data Used
The dashboard uses the dataset:
`sales.csv`
